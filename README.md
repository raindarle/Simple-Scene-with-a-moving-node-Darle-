Simple Scene with a Moving Node
Description

This beginner-level Godot project demonstrates a 2D “Hello World” label moving horizontally across the screen.

Extra features:

The label shakes slightly when it hits the left or right edges.

Movement speed and shake effect are adjustable via the Inspector.

Project Files

hello_world.gd – GDScript controlling label movement and shake.

main.tscn – Main scene with the Label node.

icon.svg – Project icon.

project.godot – Project configuration.

(Other files like .import and .gd.uid are auto-generated by Godot.)

Features

Moving “Hello World” label.

Edge collision detection with bounce.

Shake effect on impact.

Instructions to Run

Open Godot Engine v4.5.1.

Open the project folder containing project.godot.

Run the main.tscn scene.

Adjust speed, shake_strength, or shake_duration in the Inspector if needed.
